# Git and GitHub Notes

## What is Git?
Git is a version control system that helps developers track changes in their projects.

## What is GitHub?
GitHub is an online platform used to store Git repositories and collaborate with others.

## Common Git Commands
- git clone
- git status
- git add .
- git commit -m "message"
- git push

## Why Branches Matter
Branches allow developers to work on new features without affecting the main project.
